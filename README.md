# 使用说明
这个项目大致是用opencv识别图片并提取坐标，然后坐标绘制多段线，绘制方式又有两种。使用时只用改变文件路径即可。
## ezdxf
基于ezdxf库实现矢量图绘制，导入CAD后用打断命令即可

全平台通用，无论时Linux还是Windows
## pywin32
利用pywin32调用CAD的COM接口，实现实时绘制，只适用于Windows，如果要实现全平台通用可以尝试comtypes，但是学习版CAD的注册表有问题，comtypes可能用不了

实用性不如ezdxf，但是你可以用它在CAD里播放动画，

但是播放CAD要先进行视频识别，视频本质是许多图片，然后分别储存每一帧包含的坐标，最后在CAD中进行批量图片绘制
